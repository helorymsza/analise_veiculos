# Projeto Análise de Veículos

Este projeto consiste em um aplicativo web interativo, desenvolvido com **Streamlit**, para análise exploratória de dados de anúncios de veículos. A aplicação permite visualizar e compreender melhor as características dos veículos anunciados a partir do conjunto de dados (`vehicles_us.csv`).

### Objetivo do Projeto
O objetivo principal é facilitar a análise visual dos dados, permitindo identificar padrões, distribuições e relações entre variáveis, como preço, quilometragem, ano do veículo e tipo de combustível, de forma simples e intuitiva.

### Funcionalidades
- Visualizações interativas (ex: histogramas, gráficos de dispersão) criadas com **Plotly**.
- Interface simples e acessível via navegador.
- Suporte para execução no Render (deploy em nuvem).

### Tecnologias Utilizadas
- Python
- Streamlit
- Pandas
- Plotly
