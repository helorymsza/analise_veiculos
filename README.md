# Projeto Análise de Veículos

Este projeto é um aplicativo web desenvolvido com **Streamlit** que permite explorar um conjunto de dados de anúncios de veículos (`vehicles_us.csv`).

### Funcionalidades
- Visualizações interativas (ex: histogramas, gráficos de dispersão) criadas com **Plotly Express**.
- Interface simples e acessível via navegador.
- Suporte para execução no Render (deploy em nuvem).

### Como rodar localmente
1. Clone este repositório.
2. Crie e ative um ambiente virtual.
3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
